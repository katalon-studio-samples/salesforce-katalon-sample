# salesforce-katalon-sample

Introduction:

In this article we are focusing on how katalon studio helps users to automate the Salesforce applications.
Let’s begin-

Use case: 
Lead Management in Salesforce.
Case Management in Salesforce.
Account Management in Salesforce.
Dashboard Management in Salesforce.

Test scenarios: 
Login into salesforce application.
Create a fresh new lead.
Change the status of lead
Change the owner of the lead.
Search for a lead with the name.
Delete a lead.
Close the application/browser.

How to automate:
Add Capability to handle Popup(Show Notification). 
Record test script via katalon recorder.
Save test objects and test cases.
Run the test on the browser.
Verify the test result.
